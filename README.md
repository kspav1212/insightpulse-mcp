# InsightPulse MCP

AI Marketing Diagnostics MCP Server.

InsightPulse analyzes marketing performance across multiple marketing channels and provides clear diagnostics and insights.

## Data Sources

* Google Analytics (GA4)
* Google Search Console
* Google Ads
* Meta Ads (Facebook / Instagram)

## Core Capabilities

* Marketing Health Score
* Traffic anomaly detection
* SEO diagnostics
* Paid ads performance analysis
* Landing page diagnostics
* AI-generated marketing insights

## MCP Tools

The server exposes the following tools for AI agents:

get_marketing_overview
High-level marketing performance summary.

analyze_traffic_sources
Breakdown of traffic by channels.

detect_traffic_anomalies
Detect unusual spikes or drops in traffic.

analyze_seo_performance
Analyze SEO metrics and issues.

analyze_paid_ads_performance
Evaluate paid campaigns.

analyze_landing_pages
Landing page diagnostics.

generate_diagnostics_report
